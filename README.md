# Saksham Saxena - Personal Website

This repository contains the source code for my personal portfolio website, hosted on GitHub Pages. It showcases my background, projects, assignments, and interests.

## Overview

*   **URL**: [https://saksham7saxena.github.io](https://saksham7saxena.github.io)
*   **Purpose**: To present my academic and professional portfolio.
*   **Template**: Based on the *SolMusic* HTML template by Colorlib.

## Sections

*   **About Me**: Introduction and background as an engineering and design student at IIT Delhi.
*   **Assignments**: A collection of my coursework and assignments (e.g., DSL810: Prototyping in IoT).
*   **Projects**: Details about my various projects.
*   **Interests**: Design, Science & Technology, and Hobbies.
*   **Contact**: Information on how to get in touch.

## Tech Stack

*   **HTML5 & CSS3**
*   **Bootstrap**: For responsive layout.
*   **JavaScript/jQuery**: For interactivity and plugins (Owl Carousel, SlickNav).

## Running Locally

1.  Clone the repository:
    ```bash
    git clone https://github.com/saksham7saxena/saksham7saxena.github.io.git
    ```

2.  Open `index.html` in your web browser.
